# Python Scripts
---

This is a collection of random python scripts, to help me expand and showcase my Python knowledge.

## How to Use
---

If you want to run a script, simply install the requirements by executing `pip3 install -r requirements.txt` and then run a script by executing `python3 [File] [Arguments]`

## Examples
---

The following command will fetch a random Wikipedia page.
```
python3 wikipedia.py -random
```
